## Installation Guide
```bash
$ sh oneKey.sh
[NAME]
    oneKey.sh -- setup Git through one script

[SYNOPSIS]
    oneKey.sh [home | root | help]

[DESCRIPTION]
    home -- install to ~/.usr/
    root -- install to /usr/local/
```
