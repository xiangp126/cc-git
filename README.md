# Installation Guide
onekey to install git version 2.15.0, need old version git

suitable for Ubuntu and CentOS

```bash
$ sh oneKey.sh

[NAME]
    oneKey.sh -- setup Git through one script

[SYNOPSIS]
    oneKey.sh [home | root | help]

[DESCRIPTION]
    home -- install to ~/.usr/
    root -- install to /usr/local/

  ____ ___ _____
 / ___|_ _|_   _|
| |  _ | |  | |
| |_| || |  | |
 \____|___| |_|

$ sh oneKey.sh install

```
